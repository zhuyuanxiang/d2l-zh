# 计算性能
:label:`chap_performance`

在深度学习中，数据集和模型通常很大，这涉及大量计算。因此，计算性能非常重要。本章将重点介绍影响计算性能的主要因素：命令式编程、符号编程、异步计算、自动并行度和多 GPU 计算。通过研究本章，您可以进一步提高前几章中实施的模型的计算性能，例如，通过在不影响准确性的情况下缩短训练时间。
